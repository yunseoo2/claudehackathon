# claudehackathon

Very brief setup

1. Frontend: `yarn` then `yarn next-dev` (or `npm install` + `npm run next-dev`).
2. Backend: create a venv, activate it, then `pip install -r requirements.txt` and run `python -m uvicorn api.index:app --reload`.

Open the frontend at `http://localhost:3001` if port 3000 is in use.
